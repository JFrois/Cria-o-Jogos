# Cria-o-Jogos
Projeto em Python, desenvolvi um programa de jogos com dois jogos: Jogo da Forca e Jogo de Adivinhação.

Jogo de Adivinhação: Implementei um sistema de pontuação que leva em consideração as chances restantes, subtraindo essa pontuação do total à medida que o jogo é jogado.  
Jogo da Forca: Trata-se de um jogo simples no qual o usuário tem apenas 7 chances para adivinhar a palavra secreta. O jogo conta com 129 palavras que são selecionadas aleatoriamente para proporcionar diversão variada.
